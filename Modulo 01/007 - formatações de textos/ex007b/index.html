Nesta frase, temos um termo em ênfase usando a tag <em> (semântica). Aqui usamos o atalho Crtl + Shift + P, depois digitamos P, e então, selecionamos as tags desejadas como <EM>, <I>, <P>, ou qualquer outra tag desejada. Já a tag EM possui um significado para o html, ela é semântica

Prefira usar tags com semântica no html, formas = CSS sentido ou semântica = html
